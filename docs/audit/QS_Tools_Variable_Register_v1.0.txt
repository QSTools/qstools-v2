# QS Tools — Variable Register v1.0

STATUS: INITIAL PLACEHOLDER

PURPOSE

This file is the master register of QS Tools variables.

Each variable should record:

- variable_name
- owning_module
- owning_file
- variable_type
- calculation_source
- downstream_consumers
- must_balance_to_pnl
- test_required
- notes

INITIAL MODULES TO REGISTER

1. Labour
2. Employee Overheads
3. Assets
4. General Overheads
5. P&L / Revenue / COGS
6. Cost Summary
7. Recovery Summary
8. Cost Allocation
9. Recovery Outcome
10. Budget / Business Outcome later

CORE PRINCIPLE

QS Tools must not trust itself by default.

Every major output must be:

- calculated
- traced
- validated
- reconciled
- explained
