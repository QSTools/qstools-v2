# QS Tools — Recovery Outcome Page (v1.0 — LOCKED)

STATUS: DECISION LAYER

---

## PURPOSE

Recovery Outcome combines recovery strategy and structural readiness into a clear business decision.

It answers:

* does this model work
* what is the main constraint
* what should change next

Recovery Outcome does NOT:

* calculate costs
* calculate recovery shares
* perform structural validation
* calculate pricing

It is a pure decision and interpretation layer.

---

## CORE DECISION FLOW (LOCKED)

### 1. Invalid share model

If:

* share_not_balanced = true

Then:

* outcome_status = not_viable
* primary_constraint_key = share_model_invalid

### 2. No recoverable labour base

If:

* no_productive_output = true
  OR
* labour_recovery_without_labour = true

Then:

* outcome_status = not_viable
* primary_constraint_key = no_recoverable_base

### 3. Recovery strategy requires structure but structure fails

If:

* active_recovery_model = asset_driven OR hybrid
  AND
* structure_valid = false

Then:

* outcome_status = not_viable
* primary_constraint_key = structure_incomplete

### 4. Asset strategy without assets

If:

* asset_share_percent > 0
  AND
* asset_recovery_without_assets = true

Then:

* outcome_status = not_viable or marginal
* primary_constraint_key = asset_strategy_without_assets

### 5. Weak structure

If:

* structure_valid = true
  BUT
* staff_coverage_percent < 70
  OR
* asset_coverage_percent < 70
  OR
* group_coverage_percent < 70

Then:

* outcome_status = marginal
* primary_constraint_key = weak_structure

### 6. Healthy state

If none of the above blocking conditions apply:

* outcome_status = viable
* primary_constraint_key = healthy

## RECOMMENDED ACTION RULE

Recommended action must be:

* short
* direct
* actionable

Examples:

* Fix recovery shares to total 100%.
* Add productive labour capacity before relying on labour recovery.
* Complete asset ↔ staff links and operational groups.
* Reduce asset recovery share or introduce assets.
* Strengthen structural coverage before scaling.
* Current model is structurally and strategically viable.
