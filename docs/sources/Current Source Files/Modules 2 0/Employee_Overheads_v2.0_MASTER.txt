# Employee Overheads (v2.0 — MASTER)

MODULE TYPE
Input module

PURPOSE
Captures staff-linked non-wage overhead costs.

SYSTEM ROLE
- extends People Cost outside Labour
- remains separate to preserve transparency

DOES NOT
- modify Labour calculations
- merge into wages
- perform allocation or recovery

INPUTS
- staff-linked overhead values

OUTPUT CONTRACT (LOCKED)

Per staff:
- staff_id
- employee_overheads_total_annual

Aggregate:
- total_employee_overheads_annual

CORE RULES
- must join to Labour via staff_id only
- must not rebuild overhead totals downstream
- must remain a separate visible People Cost component

INTEGRATION
Feeds:
- Cost Summary (People Cost)

STATUS
v2.0 — wired and complete
