# 📄 QS Tools — **Business Snapshot + Modelling Page (v1.0 — LOCKED)**

---

## 🎯 PURPOSE

This page combines:

* **Revenue (actual trading)**
* **Cost Summary (true cost structure)**

to produce:

👉 **a complete, real-world business snapshot**
👉 and provide the **foundation for modelling future performance**

---

## 🧠 SYSTEM ROLE (CRITICAL)

This page is a:

👉 **Reality + Modelling Bridge Layer**

It sits **after Cost Summary** and before any execution logic.

---

## 📍 SYSTEM POSITION

```
P&L (assignment)
↓
Revenue (trading reality)
↓
Cost Summary (true cost)
↓
Business Snapshot + Modelling  ✅ (THIS PAGE)
↓
Revenue Engine (future)
```

---

## 🔒 CORE PRINCIPLE

> Reality is locked.
> The model is an overlay.
> The outcome is derived.

---

## 🧩 WHAT THIS PAGE DOES

It answers:

1. **What did your business actually do?**
2. **What does your business actually cost?**
3. **Did it work?**
4. **What do you need to change?**

---

## 🚫 WHAT THIS PAGE MUST NOT DO

* ❌ No upstream calculation rebuilds
* ❌ No direct P&L editing
* ❌ No Cost Summary logic duplication
* ❌ No execution modelling (capacity, jobs, etc.)
* ❌ No scenario persistence (v1)

---

## 📥 INPUT CONTRACTS (LOCKED)

### From Revenue (P&L-derived)

* `total_revenue`
* `materials_cost`
* `subcontract_cost`
* `gross_profit`

---

### From Cost Summary

* `total_people_cost_annual`
* `total_asset_cost_annual`
* `total_business_overheads`
* `total_cost_burden`
* `required_revenue`
* `required_recovery_rate`
* `total_productive_output`

These must be consumed exactly as exposed.
No recalculation allowed.

---

## 🧮 CALCULATIONS (LOCKED)

### Reality Layer

```
actual_net_position =
total_revenue - total_cost_burden
```

```
recovery_gap =
total_revenue - required_revenue
```

---

### Modelling Layer (User Controlled)

Inputs:

* `target_profit`
* optional: `target_profit_percent`

---

### Derived Outputs

```
target_revenue =
total_cost_burden + target_profit
```

```
target_recovery_rate =
target_revenue / total_productive_output
```

```
revenue_gap_to_target =
target_revenue - total_revenue
```
