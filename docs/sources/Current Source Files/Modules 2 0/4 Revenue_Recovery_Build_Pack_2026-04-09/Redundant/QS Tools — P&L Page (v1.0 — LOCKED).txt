MODULE: P&L PAGE
VERSION: v1.1
STATUS: LOCKED

PURPOSE:
To present a true accounting-aligned Profit & Loss statement using structured inputs,
while acting as the baseline truth of business financial performance.

This page answers:
"Is the business actually profitable based on real financial data?"

---

CORE PRINCIPLE:

This is NOT a modelling page.

This is:
→ A structured financial statement (truth layer)
→ Based on real or input P&L values
→ Used to validate the recovery engine

---

STRUCTURE (STRICT ORDER):

1. TRADING INCOME
   - Sales
   - Other direct revenue (if applicable)

   OUTPUT:
   → Total Trading Income

---

2. COST OF SALES
   (Direct job-related costs)

   Examples:
   - Materials (Concrete, Steel, Fill)
   - Subcontract Labour
   - Equipment Hire
   - Cartage
   - Waste Disposal
   - WIP Adjustments

   OUTPUT:
   → Total Cost of Sales

---

3. GROSS PROFIT

   CALC:
   Gross Profit = Trading Income - Cost of Sales

---

4. OTHER INCOME

   Examples:
   - Vehicle Contributions
   - Rebates
   - Misc income

   OUTPUT:
   → Total Other Income

---

5. OPERATING EXPENSES

   (Business overheads)

   Examples:
   - Wages & Salaries
   - ACC / KiwiSaver
   - Insurance
   - Admin / Accounting
   - Vehicle expenses
   - Office costs
   - Subscriptions
   - Interest

   OUTPUT:
   → Total Operating Expenses

---

6. NET PROFIT

   CALC:
   Net Profit = Gross Profit + Other Income - Operating Expenses

---

DATA BEHAVIOUR:

- All values are:
  → Annual
  → Currency ($)
  → Rounded to whole dollars (no cents)

- User can:
  → Input manually OR
  → Import from accounting system (future)

---

RELATIONSHIP TO SYSTEM:

INPUT SOURCE:
- External accounting system OR manual input

DOES NOT DEPEND ON:
- Labour module
- Asset module
- Cost summary
- Recovery engine

THIS IS A PARALLEL TRUTH SOURCE

---

OUTPUT USAGE:

This feeds into:
→ Recovery Outcome (comparison layer)
→ Decision Engine

Used to answer:
"Does our model match reality?"

---

KEY INSIGHT:

This page provides:

→ Actual Revenue
→ Actual Cost Structure
→ Actual Profit

Which allows comparison against:

→ Modelled recovery requirements

---

NON-GOALS:

This page does NOT:
- Calculate charge-out rates
- Allocate cost
- Model productivity
- Drive pricing

---

UI NOTES:

- Clean accounting-style layout
- Grouped sections with subtotals
- Clear visual hierarchy:
  Income → Cost → Profit → Expenses → Net Profit
