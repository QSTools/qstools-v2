QS Tools — Volume Rate Page (v1.0 — LOCKED)
PURPOSE

The Volume Rate Page models rate-based businesses that sell work by:

tonne
cubic metre
load
other bulk production unit

Examples:

earthworks
cartage
hardfill supply and install
excavation and removal
production-machine output businesses
SYSTEM ROLE (CRITICAL)

This page is a:

volume-based production rate model page

It must:

support tonne and m3 pricing
support minimum charges
support small-job uplift
support expected annual volume
expose expected revenue / recovery contribution

It must NOT:

rebuild asset cost logic
rebuild labour cost logic
rebuild materials cost logic
become a dispatch/job management tool

CORE RULES
must allow tonne and m3 mode
must clearly explain that small jobs often require materially higher effective rates
must remain simple in v1.0
must focus on annual recovery modelling, not quoting detail
DEFINITION

The Volume Rate Page models businesses that recover cost through tonne-, m3-, or load-based pricing and tests whether expected production volume and unit rates create enough annual recovery contribution.
