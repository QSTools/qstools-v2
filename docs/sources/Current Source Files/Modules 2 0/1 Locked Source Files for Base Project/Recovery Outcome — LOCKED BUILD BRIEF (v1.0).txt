## CORE DECISION ENGINE

---

### STEP 1 — HARD FAIL: INVALID SHARE MODEL

```
IF share_not_balanced = true
→ outcome = not_viable
→ constraint = share_model_invalid
```

---

### STEP 2 — HARD FAIL: NO LABOUR BASE

```
IF no_productive_output = true
OR labour_recovery_without_labour = true
→ outcome = not_viable
→ constraint = no_recoverable_base
```

---

### STEP 3 — HARD FAIL: STRUCTURE REQUIRED BUT INVALID

```
IF active_recovery_model = asset_driven OR hybrid
AND structure_valid = false
→ outcome = not_viable
→ constraint = structure_incomplete
```

---

### STEP 4 — ASSET STRATEGY WITHOUT ASSETS

```
IF asset_share_percent > 0
AND asset_recovery_without_assets = true
→ outcome = not_viable OR marginal
→ constraint = asset_strategy_without_assets
```

---

### STEP 5 — STRUCTURE WEAK (NOT FAILING)

```
IF structure_valid = true
BUT coverage is weak (< thresholds)
→ outcome = marginal
→ constraint = weak_structure
```

Threshold guideline:

* staff_coverage < 70%
* asset_coverage < 70%
* group_coverage < 70%

---

### STEP 6 — HEALTHY

```
IF no blocking issues
→ outcome = viable
→ constraint = healthy
```
