# Assets Module (v2.0 — MASTER)

MODULE TYPE
Input module

PURPOSE
Captures capital cost structure and produces asset-level cost outputs.

SYSTEM ROLE
- standalone cost module
- provides business cost inputs

DOES NOT
- use staff_id
- assign assets to labour
- perform allocation
- perform recovery logic

INPUTS
- purchase_price
- interest_rate
- finance_term_years
- running costs (monthly)

OUTPUT CONTRACT (LOCKED)
- finance_cost_annual
- running_cost_annual
- total_asset_cost_annual

CORE RULES
- depreciation is excluded (cash model only)
- asset_id is system-owned
- downstream must not rebuild asset cost
