## IMPORTANT COST SUMMARY RULE

Cost Summary remains a pure cost layer.

This means Cost Summary:

* consumes Labour outputs
* consumes Employee Overheads outputs
* consumes Assets outputs
* consumes General Overheads outputs
* does NOT consume Recovery Summary outputs
* does NOT consume Cost Allocation outputs
* does NOT consume Recovery Outcome outputs

---

## IMPORTANT RECOVERY OUTCOME RULE

Recovery Outcome:

* consumes Recovery Summary outputs
* consumes Cost Allocation outputs
* does NOT rebuild cost logic
* does NOT rebuild recovery logic
* does NOT rebuild structural validation

Recovery Outcome is the decision layer only.

## DEFINITION

This pack represents the fully aligned QS Tools source baseline with a separated:

cost layer → recovery layer → structure layer → decision layer

so each module has a clear role, stable contract, and protected ownership boundary.
