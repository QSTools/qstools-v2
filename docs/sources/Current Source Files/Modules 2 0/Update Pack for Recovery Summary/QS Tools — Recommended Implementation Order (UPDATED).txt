PHASE 1 — FINALISE COST SUMMARY INPUTS

1. Wire Employee Overheads into Cost Summary

5. Confirm Total Cost Burden and Required Recovery

PHASE 2 — RECOVERY LAYER
6. Build Recovery Summary module

8. Validate recovery distribution calculations
9. Confirm labour-only default behaviour

---

PHASE 3 — STRUCTURAL VALIDATION
10. Build Cost Allocation v2.1
11. Wire Cost Allocation to:
- Recovery Summary
- Labour
- Assets
12. Validate:
- structural links
- operational groups
- coverage metrics

---

PHASE 4 — DECISION LAYER
13. Build Recovery Outcome module
14. Wire Recovery Outcome to:
- Recovery Summary
- Cost Allocation
15. Validate decision states:
- viable
- marginal
- not_viable
16. Confirm full flow:
Cost Summary → Recovery Summary → Cost Allocation → Recovery Outcome

---

PHASE 5 — BUDGET (FUTURE)
17. Wire Budget to Cost Summary outputs
18. Extend Budget using Recovery + Allocation + Outcome context later if explicitly locked

---

## RULE

Each phase must be locked before moving forward.

Do NOT:

* build Cost Allocation before Recovery Summary
* build Recovery Outcome before Cost Allocation is validated
* build Budget before Recovery Outcome is validated
