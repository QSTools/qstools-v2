## SYSTEM ROLE CLARITY (LOCKED)

### Cost Summary

* Aggregates all business costs
* Produces total_cost_burden
* Produces required_recovery_rate
* Defines WHAT the business costs

### Recovery Summary

* Consumes Cost Summary outputs only
* Defines recovery strategy
* Distributes recovery across labour, assets, and overhead absorption
* Defines HOW the business recovers cost

### Cost Allocation

* Consumes Recovery Summary outputs
* Consumes Labour + Asset identities
* Defines capability links and operational groups
* Validates structural readiness
* Defines CAN it be delivered

### Recovery Outcome

* Consumes Recovery Summary outputs
* Consumes Cost Allocation outputs
* Produces a business decision
* Defines SHOULD you run it
