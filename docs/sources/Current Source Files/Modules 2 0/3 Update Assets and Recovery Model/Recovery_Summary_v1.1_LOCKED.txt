NEW MODEL

Recovery shares are derived from productive cost base.

labour_base =
sum(productive labour cost rates)

asset_base =
sum(true asset cost per hour for productive assets)

total_base =
labour_base + asset_base

labour_share_percent =
labour_base / total_base

asset_share_percent =
asset_base / total_base

overhead_share_percent = 0 (default)
