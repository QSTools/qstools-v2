## OBJECTIVE

Build the Cost Summary wiring layer so the Cost Summary page can consume live upstream module data and convert it into the locked Cost Summary page structure.

This phase is specifically about:

* wiring live upstream hook outputs into Cost Summary
* shaping upstream data inside Cost Summary selectors

## CALCULATION RULES

All calculations must use locked formulas:

labour_cost_per_productive_hour = labour_cost_rate + overhead_per_productive_hour

total_labour_cost_annual = sum(labour_cost_per_productive_hour × productive_hours)

total_productive_output = sum(productive_hours)

total_cost_burden = total_labour_cost_annual + total_asset_cost_annual + total_general_overheads

required_revenue = total_cost_burden

required_recovery_rate = required_revenue / total_productive_output
