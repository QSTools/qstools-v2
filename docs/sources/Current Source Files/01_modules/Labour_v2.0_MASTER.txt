# Labour (v2.0 — MASTER)

MODULE TYPE
Input module.

PURPOSE
Labour is the identity owner and labour cost engine.

OWNS

- staff_id
- working time structure
- pay inputs
- entitlements
- employer contributions
- productive hour logic
- labour commercial outputs

DOES NOT

- include employee overheads
- include asset costs
- include business overheads
- include allocation logic

LOCKED OUTPUTS

- staff_id
- paid_hours_per_year
- non_productive_paid_hours
- productive_hours
- labour_cost_rate
- charge_out_rate
- loaded_labour_cost_rate
- productive_labour_cost_rate
- minimum_charge_out_rate
- actual_margin_percent
- margin_gap
- profit_per_hour
- above_recovery
- employer_kiwisaver_gross
- esct_amount
- total_employer_contribution_cost
- gross_wages_annual
- entitlements_annual

RULE

Cost Summary must consume these outputs directly  
and must NOT rebuild them from labour_rate.
