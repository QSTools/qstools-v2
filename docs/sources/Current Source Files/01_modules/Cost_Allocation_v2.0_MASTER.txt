# Cost Allocation (v2.0 — MASTER)

MODULE TYPE
Structural module.

PURPOSE
Defines how the business is structured to recover cost.

DOES

- sets revenue_model
- defines labour ↔ asset links
- outputs structural context

DOES NOT

- calculate costs
- calculate recovery
- calculate pricing

REVENUE MODELS

- labour_only
- asset_driven

LOCKED OUTPUTS

- active_revenue_model
- active_asset_labour_links[]
- linked_staff_count
- linked_asset_count
- unlinked_active_staff_count
- duplicate_link_warnings[]

RULE

Cost Summary may DISPLAY this structure  
but must NOT interpret recovery.

Recovery logic belongs in the Recovery Layer.
