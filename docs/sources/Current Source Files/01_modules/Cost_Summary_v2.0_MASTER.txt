# Cost Summary (v2.0 — MASTER)

MODULE TYPE
Summary module.

PURPOSE
Cost-of-business layer of QS Tools.

DOES

- aggregates all cost inputs
- calculates total business burden
- expresses cost per productive output

DOES NOT

- perform pricing
- interpret recovery
- assume revenue model

INPUTS

Labour:
- staff_id
- productive_hours
- gross_wages_annual
- entitlements_annual
- employer_kiwisaver_gross
- esct_amount

Employee Overheads:
- staff_id
- employee_overheads_total_annual
- total_employee_overheads_annual

Assets:
- total_asset_cost_annual

General Overheads:
- total_general_overheads

Cost Allocation (display only):
- active_revenue_model
- linked counts

PEOPLE COST

total_people_cost_annual =
gross_wages_annual
+ entitlements_annual
+ employer_kiwisaver_gross
+ esct_amount
+ employee_overheads_total_annual

BUSINESS COST

total_business_cost_annual =
total_asset_cost_annual
+ total_general_overheads

TOTAL BURDEN

total_cost_burden =
total_people_cost_annual
+ total_asset_cost_annual
+ total_general_overheads

OUTPUTS

- total_people_cost_annual
- total_business_cost_annual
- total_cost_burden
- required_revenue
- required_recovery_rate

CRITICAL RULE

These outputs are mechanical only  
NOT pricing or recovery decisions

NOTICE

"This is a cost-of-business view only"
