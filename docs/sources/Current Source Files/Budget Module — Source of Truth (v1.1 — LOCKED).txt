# Budget Module — Source of Truth (v1.1 — LOCKED)

---

## Purpose

The Budget Module is the forecast vs actual performance layer of QS Tools.

---

## 🔒 SYSTEM ROLE

forecast + comparison layer

---

## 📥 Inputs

### From Cost Summary

* total_cost_burden
* required_revenue
* required_recovery_rate
* total_productive_output
* total_labour_cost_annual
* total_asset_cost_annual
* total_business_overheads

---

### Scenario Inputs

* projected_hours
* projected_revenue (optional override)

---

## 📊 Core Calculations

### Revenue Model

if projected_revenue exists:

use projected_revenue

else:

projected_revenue =
projected_hours × required_recovery_rate

---

### Forecast Profit

forecast_profit =
projected_revenue - total_cost_burden

---

### Forecast Margin

forecast_margin =
forecast_profit / projected_revenue

---

## 📊 Outputs

* projected_revenue
* forecast_profit
* forecast_margin

---

## ⚠️ CORE RULES

* must use Cost Summary outputs only
* no duplication of upstream logic
* no identity ownership
