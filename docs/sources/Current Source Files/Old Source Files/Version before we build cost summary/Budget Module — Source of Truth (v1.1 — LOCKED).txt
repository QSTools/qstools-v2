# Budget Module — Source of Truth (v1.1 — LOCKED)

---

## Purpose

The Budget Module is the forecast layer of QS Tools.

It is responsible for:

* projecting revenue from planned productive output
* comparing forecast revenue against required recovery
* showing forecast commercial outcome using upstream system outputs

It is a forward-looking planning layer only.

---

## 🔒 SYSTEM ROLE (CRITICAL)

The Budget Module is a:

forecast layer

Position in system:

INPUT MODULES
↓
COST SUMMARY
↓
BUDGET

It must:

* use Cost Summary outputs only
* support forecast planning
* remain separate from source cost engines

It must NOT:

* calculate Labour logic
* calculate Asset logic
* calculate Overhead logic
* consume Scenario outputs in V1
* behave like a full P&L engine

## 🧠 Data Flow

Cost Summary outputs → Budget calculations → forecast outputs → UI display

No logic in UI.

---

## 📥 Inputs (Consumed Only)

### From Cost Summary

* total_cost_burden
* required_revenue
* required_recovery_rate
* total_productive_output
* total_labour_cost_annual
* total_asset_cost_annual
* total_business_overheads

### Local Forecast Inputs

* projected_hours
* projected_revenue

Rules:

* `projected_revenue` is optional
* if not provided, Budget must derive revenue using `projected_hours × required_recovery_rate`

---

## 📊 Core Calculations

### Revenue Model

If `projected_revenue` exists:

`projected_revenue`

Else:

`projected_revenue = projected_hours × required_recovery_rate`

---

### Forecast Profit

`forecast_profit = projected_revenue - total_cost_burden`

---

### Forecast Margin

`forecast_margin = forecast_profit / projected_revenue`

---

## 📤 Outputs

* projected_revenue
* forecast_profit
* forecast_margin

---

## ⚠️ CORE RULES

## 🔗 Integration Rules

### Cost Summary Module

* required upstream source
* provides all required recovery and cost baseline outputs

### Labour Scenario Layer

* must NOT feed Budget in V1

---

## 🚫 Not Allowed

* direct Labour calculations
* direct Asset calculations
* direct Overhead calculations
* P&L reporting logic
* actual-vs-budget reporting engine in this version
* scenario-driven forecast logic in this version

---

## ✅ Definition

The Budget Module is a forecast-only downstream planning layer that uses Cost Summary outputs to estimate projected revenue, forecast profit, and forecast margin without duplicating upstream commercial logic.
