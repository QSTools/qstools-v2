## v3.6 MODEL READINESS RULE

This module must consume or respect:

- model_ready
- blocking_modules
- warning_modules
- model_readiness_status

If model_ready = false:

- outputs may display for preview only
- outputs must not be treated as final
- the page must show a not trusted state
- downstream decisions must be blocked or clearly marked as not trusted

This module must not bypass Model Readiness.
