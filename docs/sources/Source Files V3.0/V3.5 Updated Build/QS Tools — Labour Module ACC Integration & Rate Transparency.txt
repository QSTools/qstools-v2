4. LABOUR CALCULATION (LOCKED)

Employer Contribution =
KiwiSaver + ESCT + ACC

ACC Work Levy =
Gross Wages × (ACC Rate / 100)

Total Labour Cost =
Gross Wages + Employer Contributions

Recovery:
Productive Labour Cost Rate =
Total Labour Cost ÷ Productive Hours
