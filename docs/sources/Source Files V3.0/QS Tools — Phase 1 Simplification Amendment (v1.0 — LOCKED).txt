# QS Tools — Phase 1 Simplification Amendment (v1.0 — LOCKED)

---

## PURPOSE

This amendment simplifies the Phase 1 setup flow by:

* removing early allocation logic
* preventing unstable live distribution of shared costs
* reducing required user inputs
* improving speed and usability

---

## PHASE 1 INPUT MODULES (LOCKED)

The only required input modules before Cost Summary are:

1. P&L
2. Labour
3. Assets

---

## CORE SYSTEM RULE

> Only assign costs at the point where they are stable.
> Shared or pooled costs must NOT be distributed during setup.

---

## SYSTEM CHANGE SUMMARY

* Staff Overheads module → REMOVED (Phase 1)
* Shared asset cost distribution → REMOVED (Phase 1)
* Labour → PURE labour cost only
* Overheads → handled later in Recovery layer

---

## RESULT

* Faster setup
* Reduced user error
* Stable early outputs
* Full system accuracy preserved downstream
