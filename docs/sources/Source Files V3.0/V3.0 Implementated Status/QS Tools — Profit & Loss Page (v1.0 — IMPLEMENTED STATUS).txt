QS Tools — Profit & Loss Page (v1.0 — IMPLEMENTED STATUS) — UPDATED

STATUS: COMPLETE — STABLE BASELINE

PURPOSE

The Profit & Loss page is the input classification layer for QS Tools.

It allows the user to:

enter their real-world P&L
classify each line into QS benchmark categories
produce structured benchmark outputs for downstream modules
SYSTEM ROLE

P&L Page = INPUT CLASSIFICATION + BENCHMARK LAYER

Flow:

INPUT → (P&L CLASSIFICATION) → MODULE SETUP → COST SUMMARY → RECOVERY → COST ALLOCATION

IMPORTANT CLARIFICATION

P&L outputs are:

benchmark values
classification guidance

They do NOT:

replace downstream module calculations
directly drive Cost Summary maths
