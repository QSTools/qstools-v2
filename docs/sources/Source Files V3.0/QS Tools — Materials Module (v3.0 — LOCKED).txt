# QS Tools — Materials Module (v3.0 — LOCKED)

STATUS: MARGIN CONTRIBUTION LAYER

---

## PURPOSE

To define how materials contribute margin toward recovery.

---

## INPUTS

* materials_cost (from P&L)
* materials_recovery_required (from Recovery Summary)

---

## CALCULATIONS

material_revenue_required =
materials_cost + materials_recovery_required

---

implied_margin =
materials_recovery_required ÷ material_revenue_required

---

## OUTPUTS

* material_revenue_required
* implied_margin

---

## RULES

* materials are not business cost
* materials contribute margin only
* no pricing engine here

---

## DEFINITION

Materials Module v3.0 defines the required margin contribution from materials without interfering with cost structure or rate modelling.
