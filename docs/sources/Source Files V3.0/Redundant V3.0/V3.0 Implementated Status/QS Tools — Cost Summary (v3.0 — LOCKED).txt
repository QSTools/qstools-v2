# QS Tools — Cost Summary (v3.0 — LOCKED)

STATUS: SINGLE SOURCE OF TRUTH

---

## PURPOSE

The Cost Summary defines:

> **What the business costs to run**

---

## INPUTS

From modules:

* labour_total
* employee_overheads_total
* assets_total
* general_overheads_total

---

## CALCULATIONS

total_cost_burden =
sum(all module totals)

---

required_revenue =
total_cost_burden

---

## OUTPUTS

* total_cost_burden
* required_revenue
* required_recovery_rate (if output known)

---

## CRITICAL RULE

> No other page may create or modify required_revenue

---

## DEFINITION

Cost Summary v3.0 is the financial truth layer of QS Tools. It defines the total business cost and required revenue baseline used by all downstream logic.
