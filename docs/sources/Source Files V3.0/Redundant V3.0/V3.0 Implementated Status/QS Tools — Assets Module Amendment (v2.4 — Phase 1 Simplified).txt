# QS Tools — Assets Module Amendment (v2.4 — Phase 1 Simplified)

---

## REMOVE

* pooled cost distribution:

  * fuel split across vehicles
  * shared maintenance allocation

---

## KEEP

Per-asset direct costs only:

* finance / lease
* depreciation
* fixed insurance
* clearly attributable running costs

---

## RULE

> Assets must only contain stable, per-unit costs.

---

## HANDLING OF SHARED COSTS

Examples:

* fuel
* fleet servicing
* shared maintenance

These must NOT be distributed per asset in Phase 1.

They remain part of:

→ General Overheads

---

## RESULT

Assets remain:

* clean
* stable
* easy to input
